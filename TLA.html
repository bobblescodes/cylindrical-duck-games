Features

The Map

The map is partically inspired by the Andes mountains with a large central mountain, and many mmore off in the distance. The player is able to build towers anywhere in the grassy area (and there maybe be some water-based towers as well). They player is free to structure their tower defenses however they see fit.

Companion Feature

At the start of the game, the player has a choice of their companion. Pictured to the left is a guanaco, which is a close relative to the llama and lives in the Andes. Each companion will provide a different special ability or stat boost. The player can name their companion and they will follow the player around.

Tower Building

So far there are 3 towers types. A simple attackig tower, a bank tower which prdouces gold, and a healing tower for the player. The healing tower will only work when the player is in range and enemies are out of range. The tower building mechanic is set on a grid with each tower taking up a 9 sqaure chunk.

Player Upgrades

The player can upgrade 4 main categories (as of now - this may develop into more attributes): health, damage, attack speed, and running speed. Each upgrade costs gold, and gets progressively more expensive as you upgrade. The menu on the left is not finalized (I still need to display the cost somewhere).

Tech Stack

I am creating TLA on the game engine Godot and using both C# and GDScript to handle the coding. I am doing all the artwork in Asesprite, and creating the sound effects and music in GarageBand. Luckily these tools allow me to keep my develop as a zero-cost project (except for the $20 Asesprite purchase).

Sophisticated Enemies

Enemies are smart enough to surround the player and avoid towers. I have also added in randomization to the enemy decisions. Any given enemy will randomly select to attack towers, the player, or the home base.

Under Development

Home Base

I've started the artwork for a 'home base' of sorts which enemies will try to destory. The base is modeled after some old Inca ruins. It will be up to the player on how they want to protect it. If the base is destroyed then the game is over.

More Tower Types

I'm developing multiple more tower types. Some of which are towers which do high close-range damage, towers which slow enemies down and redirect them, and towers which can repair or boost other local towers. As well as towers which do better against specific enemy tpyes.

More Enemy Types

I am working on a bunch more enemy types. One exmaple is a water-based enemy which is a stationary ship which does heavy damage to the home base (and when the player is in range) but can only be taken out by towers. Another example is an enemy which will deactivate towers if they get within range. And many more to come.
